Employee Attrition Analysis Using Python
Overview

This project analyzes the IBM HR Analytics Employee Attrition dataset to identify the factors affecting employee turnover. Exploratory Data Analysis (EDA) and visualizations were performed to understand employee behavior and attrition patterns.

Objectives
Analyze employee demographics and work-related factors.
Identify key reasons behind employee attrition.
Generate insights to improve employee retention.
Tools & Technologies
Python
Pandas
NumPy
Matplotlib
Seaborn
Jupyter Notebook
Data Cleaning
Verified dataset structure.
Checked missing values (none found).
Removed duplicate records (if any).
Ensured data consistency.
Key Insights
Employees working overtime are more likely to leave.
Lower monthly income is associated with higher attrition.
Employees with fewer years at the company show higher turnover.
Job satisfaction significantly impacts employee retention.
Conclusion

This analysis provides valuable insights into employee attrition and highlights the importance of factors such as overtime, salary, experience, and job satisfaction. These findings can help organizations develop effective strategies to improve employee retention.
